# University Face Recognition Attendance System

## Project Overview
This project is a comprehensive university attendance system featuring an Admin Panel for managing academic structure, users, and settings, and currently in development dashboards for Students and Teachers.

## Technologies Used
- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Shadcn UI
- **Backend** (Planned): Python/FastAPI (for face recognition integration)

## Features
- **Role-Based Access Control**: Admin, Teacher, Student roles.
- **Admin Panel**:
    - Dashboard with statistics.
    - Academic Structure Management (Departments, Batches, Sections, Subjects).
    - User Management.
    - Routine & Attendance Views.
    - System Settings.
- **Hierarchical Filtering**: Cascading filters for batches and sections.

## Setup
1. Clone the repository.
2. `npm install`
3. `npm run dev`
